A propos

Le secteur de l'alimentation est responsable d'un quart des emissions des gaz à éffet serre (GES) à l'échelle mondiale1.
Grâce au calculateur myCookPrint, on peut estimer l'empreinte carbone d'un répas, y compris la production, la transformation, le transport des aliments jusqu'à la préparation des mets.

Le calculateur myCookPrint utilise deux bases de données distribuées par le governemment français en open data (Licence Ouverte):

  • la base de données AGRIBALYSE® (v. 3.0.1 2020) produite par l'Agence de la Transition Ecologique française (ADEME) fournit l'analyse environnementale des aliments les plus comumuns;
  • la base de données CIQUAL (v. 2020) produite par l'Agence nationale (française) de sécurité sanitaire de l'alimentation, de l'environnement et du travail (ANSES) fournit les informations sur la composition des aliments.

Les résultats sont valides pour des produits achetés et consommés en France. Cependant, ils peuvent être utilisés comme un premier indicateur dans d'autres pays européens.

Le calculateur utilise aussi les facteurs d'emissions (CO2e/KWh) des sources énergétiques fournis par l'Agence Europeenne de l'Environnement (CO2 Intensity of Electricity generation (2017), la Base carbone® (v 19.0 4/12/2020) de l'ADEME pour la France, et l'Office Fédéral de l'Environnement (OFEN) (2014) pour la Suisse.

Le code source ainsi que les données sont disponibles dans le repository Github du projet.
